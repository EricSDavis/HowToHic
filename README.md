# HowToHic

A Quarto presentation about how to interpret Hi-C plots.

## How to run or edit (locally)

1. Clone repository
    ```bash
    git clone https://github.com/EricSDavis/HowToHic.git
    ```

2. Build and run container
    ```bash
    make up
    ```

3. Open browser to http://localhost:8787

4. Close when finished
    ```bash
    make down
    ```
